# Sky Era

> [!NOTE]
> ***This is an early alpha build - the game is still in active development.***

**The game is a singleplayer Action RPG set in an Asian fantasy world, where you master martial arts, grow in power through experience and skill, and face increasingly powerful enemies.**

## Gameplay

- Dynamic combat system
- Character progression through mastering techniques and fighting styles
- Experience-based leveling system
- Skill & Attribute progression system
- Inventory system
- Quest system
- Save/load support

---

## How to Play

1. Go to the **[Releases Section](https://github.com/most1414/SkyEra/releases)**.
2. Download the latest `SkyEra.rar` file.
3. Extract the archive and run `SkyEra.exe`.

---

## Controls

- **Movement:** Point & Click
- **Perspective:** Fixed Camera
- **Keyboard shortcuts:** Available in-game - click the yellow ***?*** button in the bottom left corner
